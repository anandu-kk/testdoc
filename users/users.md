# VisionX User Access Overview

VisionX provides different features depending on your role within the system.  
There are two primary roles:

- **User**
- **Administrator**

Each role has its own dashboard and set of capabilities.

## User Access

Regular users have access to monitoring and analytics features focused on viewing live feeds and understanding detected events.

**View full User Features:**  
[User Dashboard](regular_user.md)

## Admin Access

Administrators have full control over the VisionX platform, including device setup, model training, and more.

**View full Admin Features:**  
[Admin Dashboard](admin.md)

## Choosing Your Role

Your available features are determined by the credentials assigned to you.  
If you require additional access, please contact your system administrator.
