# User Dashboard

As a regular user, you have access to the **Live Monitor** feature to view and analyze video feeds.

## Live Monitor

The Live Monitor is your main interface for viewing and analyzing video feeds in real time.

### Camera Feeds
- View live feeds from all connected cameras  
- Switch between different cameras  
- Adjust view settings for better visibility  

### AI Detection
- AI models automatically detect and highlight:
  - Objects  
  - People  
  - Events of interest  
- Detections appear directly on the video feed  

### Event Logs
- Access logs of all detected events  
- View timestamps and event details  
- Filter or review historical detections  

### Analytics
- View statistics and analytics related to detections  
- Includes charts, summaries, and basic reports  

## Getting Started as a User

1. Log in with your user credentials  
2. You will be automatically directed to the **Live Monitor**  
3. Select a camera feed to view  
4. Watch the feed for any detected events  
5. Check the **Event Logs** for a history of detections
