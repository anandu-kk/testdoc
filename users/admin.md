# Admin Dashboard

As an administrator, you have access to **all features** of the VisionX system.

## Available Modules

### Home Dashboard
- Get an overview of system status  
- View recent activities and alerts  

### Models
- Manage available pre-trained AI models  

### Deployment
- Deploy AI models to different devices  

### Train Model
- Train custom AI models using your datasets  

### Custom Models
- Manage your custom-trained model versions  

### Infer Models Video
- Test pre-trained models on uploaded videos  

### Infer Custom Model Video
- Test custom-trained models on uploaded videos  

### Live Monitor
- Monitor real-time video feeds  
- Access AI detection features  

### Forecasting
- Predict future events or trends  
- Use time-series forecasting models  

### Device Management
- Add, edit, and manage connected cameras and devices  

### Log Viewer
- View detailed system logs and events  
- Monitor system health and activity  

## Getting Started as Admin

1. Log in with your admin credentials  
2. Explore the **Home Dashboard** for a system overview  
3. Visit the **Models** section to view pre-trained models  
4. Set up devices under **Device Management**  
5. Configure the **Live Monitor** to start monitoring cameras
